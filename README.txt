Sign Language Detection using Python, TensorFlow, and OpenCV

Steps:
1. Download and unzip the ASL MNIST dataset from: https://www.kaggle.com/datamunge/sign-language-mnist
2. Place sign_mnist_train.csv and sign_mnist_test.csv in this folder.
3. Run model.py to train the model and save it as sign_language_model.h5.
4. Run detect.py to detect signs in real-time using your webcam.
